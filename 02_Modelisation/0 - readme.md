# Modélisation des données

Ce dossier présente le modèle de données conçu pour l’analyse de performance.

## Modèle

- Schéma en étoile
- Tables de faits et dimensions
- Relations optimisées pour l’analyse
- Les formules utlisées pour effectuer les analyses

## Granularité

Analyse au niveau transactionnel permettant une exploration par :
- Produit
- Pays
- Segment client
- Période

## Objectif

Garantir un modèle performant, cohérent et orienté analyse décisionnelle.
Un schéma visuel du modèle est fourni dans ce dossier.
